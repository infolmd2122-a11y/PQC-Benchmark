# PQC Benchmark

Comparative performance evaluation between classical cryptography and post-quantum cryptography.

## Algorithms
- RSA-2048
- ECC-P256
- Kyber512
- Dilithium

## Metrics
- Key generation time
- Key sizes
- Signature sizes
- Memory usage

## Technologies
- Python
- liboqs
- cryptography
- pandas
- matplotlib
